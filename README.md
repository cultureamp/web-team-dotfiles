# Icelab Dotfiles

## Installation

```sh
brew tap thoughtbot/formulae
brew install rcm

git clone https://github.com/icelab/dotfiles.git ~/.dotfiles-icelab

rcup -d ~/.dotfiles-icelab  -x README.md -x LICENSE
```
